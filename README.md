# Open Collective Trends

Analyze and compare funding between collectives on [Open Collective](https://opencollective.com).

NOTE:

- It only supports USD.
- The page may hang when processing large datasets, e.g. babel and webpack.
- Data processing isn't optimized well when adding new collectives to the comparison.
